# 🎨 Assignment 2 - CSS Portfolio Website

## 📌 Project Title

**Personal Portfolio Website using HTML, CSS & Bootstrap**

---

## 📖 Description

This project is a modern **personal portfolio website** designed to showcase skills, projects, and experience.
It uses **HTML, CSS, and Bootstrap** to create a responsive and visually appealing UI.

The portfolio highlights personal details, technical skills, projects, and contact information in a structured format. 

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5

---

## ✨ Features

* Modern **dark-themed UI design**
* Responsive layout using **Bootstrap grid system**
* Sections included:

  * Hero section (Introduction)
  * About section
  * Skills section
  * Projects showcase
  * Experience section
  * Contact form
* Custom CSS effects:

  * Hover animations
  * Glow cursor effect
  * Styled buttons and cards
* Social media links (GitHub, LinkedIn)
