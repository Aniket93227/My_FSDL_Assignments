# ✈️ Assignment 5 - Travel Booking Web Application

## 📌 Project Title

**Travel Agency Management System (Full Stack Web Application)**

---

## 📖 Description

This project is a **full-stack travel booking web application** that allows users to register, login, explore trips, and manage bookings.

It integrates frontend and backend using modern technologies including authentication and database management.

The backend is built using Node.js, Express, and MongoDB for handling API requests and storing data securely. 

---

## 🛠️ Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript
* Bootstrap

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Bcrypt (for password hashing) 

---

## ✨ Features

### 🔐 Authentication System

* User registration and login
* JWT-based authentication
* Secure password hashing

### 🌍 Trip Management

* Add new trips
* View available trips
* Delete trips

### ❤️ Wishlist System

* Add/remove trips from wishlist
* Local storage support

### 📊 Dashboard Features

* View stats (total trips, bookings, etc.)
* Booking history tracking

### 🎨 UI Features

* Dark mode toggle
* Responsive design
* Animated cards and UI

### 💳 Booking System

* Book trips
* Apply promo codes
* View booking history
