# 📘 Assignment Write-up: Full Stack Travel Booking System

---

## 🎯 Aim

To develop a full-stack web application for managing travel bookings using frontend and backend technologies.

---

## 🎯 Objective

1. **Frontend Development**: To design an interactive UI using HTML, CSS, and JavaScript
2. **Backend Development**: To create REST APIs using Node.js and Express
3. **Database Integration**: To store and manage data using MongoDB
4. **Authentication System**: To implement secure login and registration using JWT

---

## 📖 Theory

### 1. HTML5

Provides structure for web pages including forms, cards, and navigation.

---

### 2. CSS3

Used for styling:

* Responsive layout
* Dark mode
* UI animations

---

### 3. JavaScript

Handles frontend logic:

* API calls using fetch
* DOM manipulation
* Event handling
* Local storage management

---

### 4. Node.js & Express

Used for backend:

* Create server
* Handle API routes
* Process client requests 

---

### 5. MongoDB & Mongoose

Database system used for:

* Storing user data
* Managing trip data
* Handling queries

---

### 6. JWT Authentication

Used for secure login system:

* Token-based authentication
* User session management

---

## ⚙️ Working

1. User registers and logs in
2. JWT token is generated and stored
3. User can:

   * Add trips
   * View trips
   * Book trips
4. Backend processes requests via APIs
5. Data is stored in MongoDB
6. UI updates dynamically

---

## 📊 Result

The system successfully performs full-stack operations including authentication, data management, and user interaction.

---

## ✅ Conclusion

This assignment demonstrates complete full-stack development including frontend, backend, database, and authentication integration.
