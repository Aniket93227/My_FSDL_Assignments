# 🏥 Assignment 3 - HealthHub Web Application

## 📌 Project Title

**HealthHub - Interactive Healthcare Web Application**

---

## 📖 Description

HealthHub is a dynamic web application designed to simulate basic healthcare services such as pharmacy management, appointment booking, fitness tracking, and BMI calculation.

This project integrates **HTML, CSS, JavaScript, Bootstrap, and Chart.js** to create an interactive and user-friendly interface.

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5
* Chart.js

---

## ✨ Features

* 🛒 **Pharmacy Cart System**

  * Add medicines to cart
  * Dynamic total calculation
  * Clear cart functionality

* 📅 **Appointment Booking**

  * Select date and time slot
  * Booking confirmation

* 📊 **Fitness Tracker**

  * Weekly steps visualization using Chart.js

* ⚖️ **BMI Calculator**

  * Calculates BMI based on height & weight
  * Displays health category

* 🔔 **Notification System**

  * Dynamic pop-up messages for user actions

* 🎨 **Modern UI Design**

  * Responsive layout
  * Hover effects and animations
