## 🚀 Getting Started

To run any assignment locally, follow these general steps:

1. Navigate to the specific assignment folder
2. Refer to the assignment’s `README.md` for detailed instructions

### ▶️ For Frontend Projects (Assignment 1–3)

* Open the `index.html` file directly in your browser

### ▶️ For Backend / Full Stack Projects (Assignment 4–6)

* Install dependencies:

  ```id="7k2nsa"
  npm install
  ```
* Configure environment variables (`.env` file if required)
* Start the server:

  ```id="8d3lqp"
  npm start
  ```

  or

  ```id="9f2mzx"
  node server.js
  ```
