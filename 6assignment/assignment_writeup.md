# 📘 Assignment Write-up: Sports Equipment Rental System

---

## 🎯 Aim

To develop a full-stack web application for renting sports equipment with booking and review functionality.

---

## 🎯 Objectives

1. To design a responsive frontend using HTML, CSS, and JavaScript
2. To develop backend APIs using Node.js and Express
3. To implement database using MongoDB
4. To create a booking system with automatic cost calculation
5. To implement user authentication using JWT

---

## 📖 Theory

### 1. Full Stack Development

Full stack development includes both frontend and backend development.

* Frontend: User interface
* Backend: Server logic and database

---

### 2. Node.js and Express

Used to create server and APIs:

* Handle HTTP requests
* Manage routes
* Connect to database 

---

### 3. MongoDB and Mongoose

Used for database:

* Store users, equipment, bookings, reviews
* Schema-based data modeling

---

### 4. Authentication (JWT)

* Token-based authentication
* Secure login system
* Maintains user sessions

---

### 5. Booking System Logic

* User selects equipment and dates
* System calculates:

  * Number of days
  * Rent cost
  * Total amount

---

### 6. Review System

* Users provide ratings and feedback
* Helps improve service quality

---

## ⚙️ Working

1. User registers and logs in
2. JWT token is generated and stored
3. User can:

   * Add/view equipment
   * Book equipment
   * Give reviews
4. Booking cost is calculated automatically
5. Data is stored in MongoDB
6. Frontend displays results dynamically

---

## 📊 Result

The system successfully performs equipment rental operations including booking, authentication, and review management.

---

## ✅ Conclusion

This assignment demonstrates an advanced full-stack application with real-world functionalities like rental booking, authentication, and rating systems.
