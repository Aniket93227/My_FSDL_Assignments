# 🏥 Assignment 1 - HTML Basics

## 📌 Project Title

**MediCare Hospital Website (HTML Basics Project)**

---

## 📖 Description

This project is a basic HTML webpage designed to represent a hospital website named **MediCare Hospital**.
It demonstrates fundamental HTML concepts along with simple internal CSS styling for a modern UI.

---

## 🛠️ Technologies Used

* HTML5
* Internal CSS

---

## ✨ Features

* Structured webpage with **header, navbar, hero section, and footer**
* Use of **basic HTML elements**
* Demonstration of:

  * Headings and paragraphs
  * Lists and navigation links
  * Tables
  * Text formatting tags
* Responsive layout using simple CSS
* Clean and modern UI design
