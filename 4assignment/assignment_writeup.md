# 📘 Assignment Write-up: Hospital Operations Dashboard

---

## 🎯 Aim

To design and develop an interactive hospital dashboard system for monitoring real-time healthcare data.

---

## 🎯 Objective

1. **Data Visualization**: To represent hospital data using charts and graphs
2. **Dynamic Interface**: To update data dynamically using JavaScript
3. **Dashboard Design**: To create a professional UI using Bootstrap and CSS
4. **Backend Integration**: To prepare system for API-based real-time data

---

## 📖 Theory

### 1. HTML5 (HyperText Markup Language)

HTML5 provides the structural foundation of the dashboard.

* Used for creating sections like cards, tables, and charts
* Organizes layout using containers and grid system

---

### 2. CSS3 (Cascading Style Sheets)

CSS is used to design and enhance the UI of the dashboard.

* Dark theme styling
* Card-based layout
* Responsive design using media queries

---

### 3. JavaScript (JS)

JavaScript is used to add dynamic functionality.

* DOM manipulation for updating values
* Event handling (buttons, dropdowns)
* Data simulation and real-time updates
* Chart rendering using libraries

---

### 4. Chart.js

Chart.js is used for visual representation of data.

* Line Chart → Admissions & Discharges
* Doughnut Chart → Bed Occupancy
* Area Chart → Disease Trends

---

### 5. Node.js & Express

Backend technologies used for server-side operations.

* API creation
* Data fetching
* Handling requests and responses

---

## ⚙️ Working

1. Dashboard loads with initial hospital data
2. JavaScript simulates real-time updates
3. Charts are rendered dynamically
4. User interactions:

   * Refresh data
   * Change time period
   * Toggle API mode
5. Updated data is displayed instantly

---

## 📊 Result

The hospital dashboard successfully displays real-time analytics with interactive charts and dynamic updates.

---

## ✅ Conclusion

This assignment demonstrates how frontend and backend technologies can be integrated to build a real-time data visualization dashboard for healthcare systems.
