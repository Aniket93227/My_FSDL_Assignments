# 📊 Assignment 4 - Hospital Operations Dashboard

## 📌 Project Title

**Hospital Operations Dashboard (Interactive Data Visualization System)**

---

## 📖 Description

This project is an interactive **Hospital Dashboard Web Application** that displays real-time and simulated hospital data such as patient admissions, bed occupancy, disease trends, and doctor schedules.

It uses modern frontend technologies along with backend-ready configuration using Node.js and Express. 

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5
* Chart.js
* Node.js
* Express.js
* MongoDB (via Mongoose) 

---

## ✨ Features

* 📊 **Patient Flow Visualization**

  * Line chart for admissions & discharges

* 🛏️ **Bed Occupancy Gauge**

  * Dynamic percentage display using doughnut chart

* 🦠 **Disease Trends**

  * Seasonal trends for Flu, Dengue, COVID

* 👨‍⚕️ **Doctor Schedule Table**

  * Displays availability and shifts

* ⏱️ **Real-Time Metrics**

  * Live time updates
  * Admissions and discharges tracking

* 🔄 **Dynamic Data Simulation**

  * Randomized real-time updates using JavaScript

* 🌐 **API Integration Ready**

  * Toggle between simulated and live API data
